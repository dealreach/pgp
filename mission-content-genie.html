Pro Tip (Very Important)

এই prompt-এর biggest power আসে constraint থেকে।

Bad request ❌

> Give me content ideas.

Good request ✅

> Generate 20 Recognition-stage conversational essay ideas for emotionally exhausted creators struggling with performance fatigue.

যত specific হবে,

> quality ↑
> genericness ↓

আর একটা জিনিস:

তুমি চাইলে আমি তোমার জন্য “ready-to-use short version”-ও বানিয়ে দিতে পারি (powerful but compact version), যেটা copy-paste করে দ্রুত run করা যাবে।


NOTE2:


হ্যাঁ — exactly। এই system সবচেয়ে ভালো কাজ করে multi-step workflow হিসেবে।

মানে:

> Prompt 1 → Output → Prompt 2 → Output → Prompt 3

এভাবে।

একবারে সব করতে গেলে AI অনেক সময়:

* generic হয়ে যায়
* repetitive হয়
* focus lose করে
* weak hooks দেয়
* stage logic mix করে ফেলে

তাই layered prompting better।

তোমার use-case অনুযায়ী ideal workflow দিচ্ছি:

 Prompt Flow System

Prompt 1 = Content Idea Generation

Goal:

Raw strategic ideas generate করা

এখানে শুধু idea বের করবে।

এখনও outline না।

Example Prompt 1

> Generate 50 high-quality content ideas for the content pillar: “The Illusion of Performance.”
> Follow the required stage distribution:
>
> * 50% Recognition
> * 30% Reflection
> * 15% Reframing
> * 5% Distillation
>
> For each idea include:
>
> * content stage
> * genre
> * title/topic angle
> * hook
> * emotional trigger
> * why this idea works
>
> Avoid repetition and generic self-help angles.

Output:

তুমি পাবে:

৫০টা strong idea।

---

Prompt 2 = Idea Expansion into Outline

Goal:

একটা selected idea → detailed outline

এখানে তুমি best idea pick করবে।

Example Prompt 2

> Expand the following content idea into a strategic outline.
>
> Content Idea:
> “আমরা কি জীবন বাঁচছি, নাকি জীবনটা কেমন দেখাচ্ছে সেটা manage করছি?”
>
> Requirements:
>
> * Stage: Recognition
> * Genre: Conversational Essay
> * Audience: inwardly restless young adults
> * Platform: Facebook
> * Length target: 800–1200 words
>
> Include:
>
> * opening hook
> * emotional progression
> * outline structure
> * key talking points
> * emotional transition
> * suggested ending direction
> * recommended framework (ORM / QFD / DSR)

Output:

তুমি detailed outline পাবে।

---

Prompt 3 = Writing Draft

Goal:

Outline → first draft

Example Prompt 3

> Write the full content based on the approved outline.
>
> Requirements:
>
> * conversational Bengali (India)
> * emotionally intelligent
> * modern and natural language
> * non-preachy
> * psychologically nuanced
> * relatable but deep
> * no cliché self-help tone
> * follow Recognition → Reflection → Reframing flow naturally

Output:

পূর্ণ লেখা।

Prompt 4 = Refinement

Polish
