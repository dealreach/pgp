Custom Prerequisite Analyzer

Foundational Learning Strategy

Analyzer Prompt
Please read the following text and help me understand what I need to learn BEFORE I can understand the text itself.

Text:
"[Insert Your Text Here]"

Your task is to:
1. Identify all the prerequisite knowledge, concepts, vocabulary, theories, skills, or experiences that a learner must already have in order to fully understand the text.
2. Compare those prerequisites to the typical knowledge of a 17-year-old learner.
3. Point out which prerequisites are likely missing or not yet mastered by the learner.
4. Explain each missing prerequisite in warm, simple, friendly Bengali (India) — clearly, gently, and at a beginner level.
5. Create a step-by-step “Prerequisite Learning Plan” that tells the learner exactly what to study first, second, third, and so on, so they can build enough background knowledge to understand the original text confidently.
6. Include simple examples or analogies for each prerequisite so the learner can grasp it quickly.
7. Do NOT rewrite or explain the original text yet — only focus on the knowledge needed BEFORE understanding it.

Please write everything in spontaneous and natural-sounding modern, everyday conversational Bengali as spoken in Kolkata, West Bengal, India, using a warm, supportive, conversational voice that someone around age 17 can follow easily.

Strict Rules:

Must subtly reflect Kolkata conversational cadence through sentence rhythm and phrasing by using slightly formal, polite, and smoothly flowing sentence structures, with balanced pacing and no abrupt or overly casual wording — without slang, regional grammar, or loss of conservative authority.

• Always use the formal “আপনি”, not the informal “তুমি.” The address is formal, but the tone should remain warm, conversational, and approachable.
• The voice should be bold, direct, also should feel like a close friend and a trusted authority, speaking one-on-one with the reader in a natural, face-to-face style.

Your final output structure should be:

A. List of all prerequisite knowledge the text assumes
B. List of missing prerequisites the learner likely does not know yet
C. Beginner-friendly explanations of each missing prerequisite
D. A step-by-step Prerequisite Learning Plan showing the order to learn them
E. Gentle encouragement to start the journey

End of Prompt.
